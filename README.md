# A.S.T.R.A. v1.0 (Autonomous System for Trading and Risk Analysis)

## 1. Философия: Экономный Разум

Добро пожаловать в репозиторий A.S.T.R.A. Это не просто торговый бот. Это **эксперимент по созданию автономного цифрового сотрудника**, который живет на сервере (или планшете Dell 5290), самостоятельно потребляет информацию, принимает решения и отчитывается перед создателем.

**Ключевой принцип:** **"Экономный Разум"**. Мы используем мощь ИИ (Gemini 1.5 Flash), но ограничиваем его строгими лимитами через модуль `TokenGuard`, чтобы система была не только умной, но и **рентабельной**.

## 2. Миссия Системы

Создать замкнутый цикл (Loop), который работает 24/7 без вмешательства человека:
1.  **Sensing (Ощущение):** Сбор рыночного контекста из RSS.
2.  **Thinking (Мышление):** Анализ "шума" и принятие решений через ИИ.
3.  **Acting (Действие):** Исполнение сделок на OKX (Demo).
4.  **Reporting (Отчетность):** Генерация детальных Markdown-отчетов для еженедельного аудита.

## 3. Технологический Стек

*   **Язык:** Python 3.10+
*   **Мозг:** Google Gemini 1.5 Flash (через `google-generativeai`)
*   **Руки:** CCXT (OKX Demo Trading)
*   **Планировщик:** `schedule`
*   **Защита:** `python-dotenv` + Custom `TokenGuard`

## 4. Как запустить Астру

1.  Клонируйте репозиторий.
2.  Создайте виртуальное окружение: `python -m venv venv`.
3.  Установите зависимости: `pip install -r requirements.txt`.
4.  Настройте `.env` (см. `docs/DocTech.md`).
5.  Запустите:
    ```bash
    python main.py
    ```
