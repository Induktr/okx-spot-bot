# Руководство Пользователя

## Введение
Этот бот предназначен для трейдеров, которым важна скорость исполнения сделок без использования веб-интерфейса биржи.

## Подготовка к работе
1.  Убедитесь, что на балансе **Futures** есть USDT.
2.  Убедитесь, что в файле `.env` прописаны корректные ключи.

## Примеры команд

### Сценарий 1: Быстрый вход в сделку
Рынок резко пошел вверх, нужно срочно купить Bitcoin.
```bash
python bot.py --action long --symbol BTC/USDT --amount 0.005

Сценарий 2: Экстренный выход (Panic Button)
Нужно срочно закрыть позицию по Ethereum, не считая объем руками.

python bot.py --action close --symbol ETH/USDT

Частые вопросы (FAQ)
В: В каком формате указывать символ?
О: Всегда используйте формат BASE/QUOTE, например BTC/USDT, ETH/USDT.
В: Что делать, если бот пишет "Insufficient Funds"?
О: Проверьте баланс Фьючерсного кошелька. Возможно, деньги лежат на Спотовом кошельке. Переведите их.
