# Интеграция с MEXC API

## 1. Протокол
Используется **REST API** биржи MEXC версии V3 (через адаптер CCXT).

## 2. Эндпоинты
Бот использует следующие методы библиотеки `ccxt`, которые маппятся на эндпоинты биржи:

| Метод CCXT | Эндпоинт MEXC (примерный) | Назначение |
| :--- | :--- | :--- |
| `fetch_balance` | `GET /api/v1/private/account/assets` | Получение баланса фьючерсов |
| `create_order` | `POST /api/v1/private/order/submit` | Создание ордера (Market) |
| `fetch_positions` | `GET /api/v1/private/position/open_positions` | Получение текущих позиций для закрытия |

## 3. Ограничения (Rate Limits)
MEXC имеет лимиты на количество запросов (обычно около 20 запросов в секунду для торговых операций).
*   **Текущая реализация:** Бот делает 1-2 запроса за запуск. Лимиты не будут превышены при ручном использовании.
