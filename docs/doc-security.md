# Политика Безопасности

## 1. Хранение Секретов
*   **Принцип:** API ключи (`API Key`, `Secret Key`) никогда не хранятся в исходном коде.
*   **Реализация:** Используется библиотека `python-dotenv`. Ключи загружаются из файла `.env`, который добавлен в `.gitignore` и не попадает в репозиторий.

## 2. Привилегии API Ключей
Для работы бота требуются минимально необходимые права.
*   ✅ **Futures Trading:** Разрешено (для открытия ордеров).
*   ✅ **Read Info:** Разрешено (для чтения баланса и позиций).
*   ❌ **Withdrawal (Вывод средств):** **КАТЕГОРИЧЕСКИ ЗАПРЕЩЕНО**. Никогда не включайте эту галочку при создании ключей для торговых ботов.

## 3. Логирование
*   Бот не пишет логи с чувствительной информацией (ключами) в файлы.
*   Вывод производится только в `stdout` (консоль) текущей сессии.
