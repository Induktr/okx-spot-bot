# Руководство для Разработчика

### Развертывание среды
1.  Используйте `venv` для изоляции зависимостей.
2.  Соблюдайте PEP-8 при написании кода.

### Расширение функционала
Если потребуется добавить **Limit Orders** (лимитные ордера):
1.  Добавьте новый аргумент `--price` в `argparse`.
2.  Добавьте новое действие `limit_long` / `limit_short`.
3.  Используйте методы `exchange.create_limit_buy_order`.

### Структура проекта
Код разделен на "Клиент" (`mexc_client.py` - логика) и "Интерфейс" (`bot.py` - ввод/вывод). Сохраняйте это разделение при добавлении новых функций. Не пишите логику `ccxt` внутри `bot.py`.
