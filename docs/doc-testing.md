# План Тестирования

## 1. Тестовое окружение
*   **Биржа:** MEXC Futures (Testnet или реальный счет с минимальным балансом 5-10 USDT).
*   **ОС:** Windows / Linux / macOS.

## 2. Сценарии Тестирования (Test Cases)

| ID | Сценарий | Входные данные | Ожидаемый результат |
| :--- | :--- | :--- | :--- |
| **TC-01** | **Проверка баланса** | `python bot.py --action balance` | Вывод в консоль суммы USDT (например, `Balance: 105.50 USDT`). |
| **TC-02** | **Открытие LONG** | `... --action long --symbol BTC/USDT --amount 0.001` | Сообщение `SUCCESS`. Позиция появляется в терминале биржи. |
| **TC-03** | **Закрытие позиции** | `... --action close --symbol BTC/USDT` | Сообщение `SUCCESS`. Позиция на бирже исчезает (или size = 0). |
| **TC-04** | **Невалидный символ** | `... --symbol FAKE/USDT` | Сообщение `ERROR: Bad Symbol` или аналогичное от CCXT. |
| **TC-05** | **Сбой сети (Симуляция)** | Отключить интернет, запустить команду | Сообщение `RETRYING...`. При восстановлении сети — `SUCCESS`. |
| **TC-06** | **Отсутствие аргументов** | `python bot.py` | Вывод справки `usage: bot.py ... error: the following arguments are required`. |
